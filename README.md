♾️ Recur

Recur is a high-performance, monochrome-styled authentication engine built with Node.js, Express, and MongoDB. It utilizes stateless JWT (JSON Web Tokens) for secure, scalable user management and is supposed to track online subscriptions(still working on it)

🛠️ Tech Stack

Backend: Node.js, Express

Database: MongoDB (via Mongoose)

Authentication: JWT, BcryptJS

Frontend: HTML5, Tailwind CSS, Phosphor Icons
